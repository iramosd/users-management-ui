# Web para gestión de usuarios

## Requisitos

Node V20 o mayor

## Configuración

Instalar dependencias:

```bash
# npm
npm install
```

## Servidor de desarrollo

Servidor de desarrollo en `http://localhost:3000`:

```bash
# npm
npm run dev -- -0
```

## Producción

Preparar para producción:

```bash
# npm
npm run build
```
